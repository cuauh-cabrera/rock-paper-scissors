# 📖 [Rock Paper Scissors] <a name="about-project"></a>

> This is a solution to the Rock Paper Scissors challenge on the Foundations Course at The Odin Project Fullstack Curriculum.

**[ Rock Paper Scissors ]** is the first JavaScript capstone project included on the Foundations Course at the The Odin Project. This project is aimed to apply the JavaScript fundamentals to code the "Rock Paper Scissors" classical game  in order to play against the computer.
